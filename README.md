## 🚀 Qu’est-ce que pinGPT ?

pinGPT est une plateforme de prototypage électronique pensée pour les débutants.

Le projet combine :
- un **PCB équipé d’un ESP32**,
- l’utilisation de **CircuitPython**,
- et un **tuteur en ligne interactif** permettant de guider l’utilisateur étape par étape.

L’utilisateur peut connecter les composants électroniques de son choix :
- LEDs,
- matrices LED,
- écrans,
- capteurs de température,
- servomoteurs,
- capteurs de distance,
- modules I2C,
- etc.

## 🧠 Le concept

pinGPT agit comme un assistant de prototypage.

Le système aide l’utilisateur à :
1. brancher correctement un composant,
2. identifier les bonnes connexions,
3. installer les bibliothèques nécessaires,
4. téléverser le code CircuitPython,
5. tester immédiatement le composant.

L’objectif est de rendre l’électronique embarquée :
- plus simple,
- plus visuelle,
- plus autonome,
- et accessible sans connaissances avancées.

---

## ⚡ Pourquoi CircuitPython ?

pinGPT utilise CircuitPython pour permettre :
- un développement rapide,
- une modification simple du code,
- une expérience proche du “plug-and-play”,
- un apprentissage plus accessible pour les débutants.

## 🛠️ Philosophie du projet

Le projet suit une logique simple :

> CONNECTER → COMPRENDRE → TESTER → CRÉER

pinGPT cherche à réduire :
- les erreurs de câblage,
- la complexité technique,
- et la peur de commencer l’électronique.
